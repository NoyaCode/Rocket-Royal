## Controls:


**Controller** (PlayStation)


Menu:
- Change Team → ```L1```/```R1```
- Ready → ```CROSS```

Game:
- Move → ```Left Joystick```
- Jump → ```CROSS```
- Aim → ```Right Joystick```
- Shoot → ```R1```
- Ultimate → ```SQUARE```

**Mouse & Keyboard**:

Menu:
- Join Lobby → ```Left Shift```
- Change Team → ```E```
- Ready → ```Space```

Game:
- Move → ```WASD```
- Jump → ```Space```
- Shoot → ```Left Mouse Button```
- Ultimate → ```Right Mouse Button```
